# Assistant permissions and preferences

Control how Assistant submits prompts, requests permission, and displays responses in the Unity Editor.

Use Assistant preferences to define how you send prompts, how Assistant requests approval before it acts in your project, and how its reasoning appears in conversations.

| Topics | Description |
| ---------- | ----------- |
| [Configure Assistant permissions and preferences](xref:preferences) | Set prompt submission behavior, permission levels, Autorun, and reasoning display. |
| [Use prompt settings menu](xref:prompt-settings) | Access prompt-level options, such as project overview refresh and other controls. |
| [Enable tooltips in Play mode](xref:tooltip) | Enable tooltips to appear in Assistant even when Play mode is activated. |
